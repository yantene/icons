yantene's icons
=============

yantene が使用しているアイコンです。

特段ライセンスの設定などはしませんが、
yantene を表す画像として使用する限り、
良識の範囲内で使用することができます。

[Releases](https://github.com/yantene/icons/releases/latest)
よりダウンロードしてください。

## 画像一覧
